# Rust OS 개발 가이드

Rust를 활용한 운영체제 작성에 대한 종합 가이드 문서입니다.

## 문서 보기

로컬에서 문서를 보려면:

```bash
# 개발 서버 실행
mkdocs serve

# 빌드
mkdocs build
```

문서는 `http://localhost:8000`에서 확인할 수 있습니다.

## 내용

- **시작하기**: 운영체제 개발의 기본 개념과 계획 수립
- **초기 설정**: 개발 환경 구축 및 필요한 도구 설치
- **개발 가이드**: 부트로더부터 커널 초기화까지 단계별 가이드

## 프로젝트 구조

```
os-make/
├── docs/          # 문서 소스 파일
├── mkdocs.yml     # MkDocs 설정 파일

```
